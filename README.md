# Проект
## Тема: Для каких типов рядов какая трансформация ряда полезнее?

### Как запустить эксперимент

1. Скачайте датасет [отсюда](https://zenodo.org/records/4656480) и распакуйте архив
3. В файле config.py поменяйте DATA_PATh на нужный
4. Запустите эксперимент с помощью команды
```bash
python <your_path_to_file>/run_experiment.py
```
Файл src/stats.csv содержит информацию о том, для какого числа рядов в кластерах какая трансформация наиболее полезна относительно какой-то метрики
